About

2048 is a video game, written by 19-year-old Italian developer Gabriele Cirulli in the JavaScript programming language. The game is a number puzzle with a field in the form of a square 4x4 cell. The goal of the game is to stack tiles with numbers to get a tile with a denomination of "2048", but you can continue playing.
